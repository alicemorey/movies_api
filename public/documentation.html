My Movie API (My FlixApp)

What does this do?

This REST API is a server-side component of a “movies” web application. The web application will provide users with access to information about different thriller and horror movies, directors and genres. Users will be able to sign up, update their personal information and create a list of their favorite horror movies. This API has URLs, returns JSON responses, and uses HTTP response codes.

User Endpoints

Request URL HTTP method Description
User login /login POST A JSON object holding Username and Password
Register new user /users POST A JSON object registering a new user
Update user information /users/update PUT A JSON object holding data about information about the user to update

Movie Endpoints

Request URL HTTP method
Return a list of all movies /movies GET
Return data about a single movie by title (description, genre, director, image URL, featured status) /movies/{Title} GET
Return data about a genre of a movie /genres/:Name GET
Return data about a director of a movie /directors/:Name GET
register a new user /users POST
add data about a users favourite movie /users/:Username/FavouriteMovie POST
